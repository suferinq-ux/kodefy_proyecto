# 🐔 Pocholo's POS

Sistema de punto de venta para pollerías. PWA moderna con gestión de inventario, ventas y reportes.

## Tecnologías

- Next.js 14 + React
- Supabase (Base de datos)
- Tailwind CSS
- Framer Motion
- 
## Funcionalidades

- Apertura y cierre de caja
- Punto de venta
- Gestión de mesas
- Vista de cocina
- Reportes y estadísticas
- Resumen por WhatsApp
